学 习 成 果 介 绍

MyCat + MySQL集群

MyCat负责MySQL读写分离，数据库集群内MySQL互为主从，从MySQL负责读，主MySQL负责写，并通过GTID控制数据同步；当发生故障，一方可以接替读写工作

Spring全家桶

后端采用 SpringBoot + SpringCloud Alibaba + SpringSecurity OAuth2 微服务框架，各服务间互不影响，一个功能故障不会影响另外一个功能。集群内包含Nacos，Gateway等组件

Vue + ElementUI + VueAdmin

前端采用 Vue + ElementUI + VueAdmin框架，可以快速迭代页面，同时支持PC，手机，IPAD等多端设备

CIDI持续集成

Kubernetes管理Docker, 同时构建Maven仓库，Harbor仓库，GitLab代码仓库，Jenkins持续集成，Nexus私服
